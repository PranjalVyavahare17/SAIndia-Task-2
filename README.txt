## Features
- Greeting label (English)
- Search bar with emoji and search button
- 5-day forecast display (dummy weather data)
- Clean and responsive Swing UI

## How to Run

1. Open terminal in this folder.
2. Compile:
   javac WeatherForecastApp.java
3. Run:
   java WeatherForecastApp

## Notes
- This version uses dummy data.
- For live forecast, OpenWeat
